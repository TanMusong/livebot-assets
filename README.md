# 🤖 LiveBot 脚本介绍

## 一、脚本简介

**LiveBot** 是一款抖音直播间智能场控助手 🎤，帮你自动搞定直播间的互动琐事——进场欢迎 👋、点赞感谢 ❤️、礼物播报 🎁、AI 聊天回复 💬，统统安排上！  
搭配可视化设置面板 ⚙️，轻轻松松就能定制属于你的互动风格 🎨。

---

## 二、核心功能

### 🎉 基础互动
- 进场欢迎消息（普通用户 + 高等级用户分别欢迎 👑）
- 点赞感谢消息 💕
- 礼物感谢消息 🎁（支持多种礼物名称解析）

### 🧠 AI 加持
- AI 智能回复 @消息 🤖，支持 8 种性格风格可选：
  😊 友好 · 🔥 热情 · ⚡ 活泼 · 😄 幽默 · 🌸 温和 · 🎉 气氛组 · ✨ 高情商 · 🎯 简洁
- 多轮对话记忆 🧩，记住和用户的聊天上下文，回复更连贯
- 支持自定义模型接入，兼容 OpenAI 接口格式 🔗

### 🛡️ 直播守护
- 直播结束自动暂停机器人 ⏸️，不再发无意义消息
- 消息冷却与发送节流 ⏱️，告别刷屏困扰
- 调试模式 🔍：先预览不发送，确认没问题再上线

### 📋 配置管理
- 每个主播独立配置，互不干扰 🎯
- 配置列表一目了然，支持快速切换和应用 🔄
- 设置即时生效 ⚡，改完不用刷新页面

### 🛠️ 实用工具
- 悬浮日志面板 📜：按模块筛选、自动跟随、一键清空
- 直观的浮窗设置面板 🎛️，所有功能触手可及

---

## 三、使用小贴士 💡

- 🔍 首次使用建议先开日志面板，看看事件捕获是否正常
- 🧪 功能可以逐个开启，先试试效果再全量启用
- 🤖 开启 AI 前记得先跑一下连接测试，确认模型没问题
- ⏱️ 冷却时间别设太短，免得消息发得太勤被限流哦
